777 /* Guarded Dwarf3 expressions, which can be linked together to form a
778  list. The payload field contains a variable length array of bytes
779  which hold the guarded expressions. The length can be inferred by
780  inspecting the payload bytes and so does not need to be stored
781  explicitly.
782 
783  Guarded-Expression format is similar but not identical to the
784  DWARF3 location-list format. The format of each returned block is:
785 
786  UChar biasMe;
787  UChar isEnd;
788  followed by zero or more of
789 
790  (Addr aMin; Addr aMax; UShort nbytes; ..bytes..; UChar isEnd)
791 
792  '..bytes..' is an standard DWARF3 location expression which is
793  valid when aMin <= pc <= aMax (possibly after suitable biasing).
794 
795  The number of bytes in '..bytes..' is nbytes.
796 
797  The end of the sequence is marked by an isEnd == 1 value. All
798  previous isEnd values must be zero.
799 
800  biasMe is 1 if the aMin/aMax fields need this DebugInfo's text_bias
801  added before use, and 0 if the GX is this is not necessary (is
802  ready to go).
803 
804  Hence the block can be quickly parsed and is self-describing. Note
805  that aMax is 1 less than the corresponding value in a DWARF3
806  location list. Zero length ranges, with aMax == aMin-1, are not
807  allowed.
808 */

811  UChar payload[0];
812  }
