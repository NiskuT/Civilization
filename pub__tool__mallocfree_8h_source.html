52 // VG_(perm_malloc) is for allocating small blocks which are
53 // never released. The overhead for such blocks is minimal.
54 // VG_(perm_malloc) returns memory which is (at least) aligned
55 // on a multiple of align.
56 // Use the macro vg_alignof (type) to get a safe alignment for a type.
57 // No other function can be used on these permanently allocated blocks.
58 // In particular, do *not* call VG_(free) or VG_(realloc).
59 // Technically, these blocks will be returned from big superblocks
60 // only containing such permanently allocated blocks.
61 // Note that there is no cc cost centre : all such blocks will be
62 // regrouped under the "perm_alloc" cost centre.
63 extern void* VG_(perm_malloc) ( SizeT nbytes, Int align );
