29 /* This file defines types and constants for the kernel interface, and to
30  make that clear everything is prefixed VKI_/vki_.
31 
32  All code is copied verbatim from kernel source files, except that:
33  - VKI_/vki_ prefixes are added
34  - some extra explanatory comments are included; they are all within
35  "[[ ]]"
36  - for some types, we only care about the size; for a few of them (big
37  ones that are painful to fully drag in here), a VKI_SIZEOF_* constant
38  is used.
39 
40  The files the code is taken from is indicated.
41 
42  Note especially that the types are not the glibc versions, many of which
43  are different to those in here.
44 
45  Also note that this file contains all the generic header info, ie. that
46  from linux/include/linux/ *.h. The arch-specific header info, eg. that
47  from linux/include/asm-i386/ *.h, is in vki-$PLATFORM.h and
48  vki_posixtypes-$PLATFORM.h. (Two files are required to avoid
49  circular dependencies between the generic VKI header and the
50  arch-specific VKI header. It's possible in the future, as more stuff
51  gets pulled in, that we might have to split files up some more to avoid
52  further circular dependencies.)
53 
54  Finally, note that it is assumed that __KERNEL__ is set for all these
55  definitions, which affects some of them.
56 */
